## Commands
- `+list` Lists all BoB servers

## Setup
- Create a new application on https://discordapp.com/developers/
- Add a bot user
- Give the bot user the following permissions
  - View Channels
  - Send Messages
- Create the following 2 environment variables
  - BOT_PREFIX
  - BOT_TOKEN
- Run `npm install`
- Run `node index.js`
